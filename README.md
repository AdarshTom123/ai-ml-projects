# AI/ML Projects

A collection of machine learning and deep learning projects covering natural language processing, computer vision, neural networks, and classification.

These projects started as my Grade 12 AI coursework and I've documented and organized them into a portfolio.

## Tech Stack
- **Languages:** Python
- **ML/DL:** scikit-learn, TensorFlow/Keras
- **NLP:** spaCy, NLTK
- **Data:** Pandas, NumPy
- **Visualization:** Matplotlib

## Projects

| Project | Description | Techniques |
|---------|-------------|------------|
| [Email Spam Classifier](./spam-classifier) | Classifies emails as spam or ham | Naive Bayes, CountVectorizer |
| [Traffic Sign CNN](./cnn-traffic-signs) | Convolutional neural network for image classification | CNN, Keras |
| [Handwritten Digit Recognition](./mlp-digit-recognition) | Classifies handwritten digits (96.67% accuracy) | MLP, scikit-learn |
| [Named Entity Recognition](./named-entity-recognition) | Extracts people, places, and organizations from text | spaCy, NLP |
| [Neural Network Basics](./neural-network-basics) | Forward and backward propagation from scratch | NumPy |
| [Car Price Predictor](./car-price-prediction) | Predicts used car prices | Decision Tree Regression |
