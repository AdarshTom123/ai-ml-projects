# Car Price Predictor

A machine learning model that predicts the selling price of used cars based on engine size and age, using Decision Tree Regression.

## How it works
1. Loads a dataset of cars with engine size, age, and selling price
2. Trains a Decision Tree Regression model
3. Predicts the price of a new car

## Tech Stack
- Python
- scikit-learn (DecisionTreeRegressor)
- Pandas

## How to run

1. Install libraries: pip install pandas scikit-learn
2. Run the script: python car_price.py
