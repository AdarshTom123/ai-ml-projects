# Neural Network Basics (Forward & Backward Propagation)

A from-scratch implementation of the core mechanics of a neural network using only NumPy — no deep learning libraries. This demonstrates the fundamental math behind how neural networks learn.

## What it shows
- **Forward propagation:** how inputs, weights, and bias combine to produce predictions (dot product + bias)
- **Backward propagation:** how gradients are calculated to understand each feature's effect on the output

## Why it matters
Instead of just calling a library like Keras, this implements the underlying operations manually — showing an understanding of what happens inside a neural network.

## Tech Stack
- Python
- NumPy
- Pandas

## How to run

1. Install libraries: pip install numpy pandas
2. Run the script: python neural_network.py
